Madison Digital Imaging Database¶

The Madison Digital Image Database is a digital media management system with sophisticated tools for discovering, aggregating, and presenting digital media in a wide variety of learning spaces.

MDID is a freely distributed, open source web application developed at James Madison University.

  • MDID3 Website: http://mdid.org
  • Official Repository: https://github.com/cit-jmu/rooibos/
  • MDID3 Mailing List: https://listserv.jmu.edu/cgi-bin/wa?SUBED1=mdidusers-l&A=1

Please join the list and introduce yourself if you end up working with MDID3.
